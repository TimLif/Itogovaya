// Разработать одностраничный сайт для рекламного агенства.

Задачи: {

- методология БЭМ и общая информация о сайте.
- псевдоэлементы и анимация CSS.
- адаптив — flex.
- адаптив — grid.
- медиа запросы для планшетов.
- медиа запросы для мобильных устройств.
- бургер-меню на JS для мобильных устройств.
- кнопка прокрутки вверх на JS.
}

Этап разработки начинался с анализа схожих элементов, для дальнейшего использования общих классов, а так же разбиения сайта на логические и удобные для верстки блоки. Веб ресурс создавался полностью на нативном html, css, js без использования каких-либо фреймворков и библиотек. В проект веб ресурса входят 1 страница в формате html, 1 страница в формате css, папка с изображениями.

При переходе по адресу https://timlif.github.io/diplom/ пользователь попадает на главную страницу сайта, на которой в шапке расположено меню для перехода на другие разделы страницы веб ресурса.

В блоке под шапкой расположены блоки с краткой информацией о компании, а так же видео.

При нажатии на кнопку «Portfolio», перебрасывает на часть страницы с информацией. На данной части страницы представлены данные об услугах. Реализована анимация css при наведении (hover) на иконки.

В веб ресурсе представлена краткая информация о сотрудниках компании в разделе «about us» и ссылкой на соц сети. При наведении на прямоугольники, появляются имена сотрудников. Адаптив сделан через flex.

Последний блок на странице ”contacts”. Реализована форма обратной связи, а так же возможность позвонить. Адаптив сделан через grid.

Реализованы медиа запросы для устройств менее 1024 пикселей

А так же медиа запросы для устройств менее 768 пикселей

В варианте для устройств размером экрана менее 768 пикселей реализовано бургер-меню.

Для удобства возвращения к началу страницы, добавил от себя кнопку прокрутки.
